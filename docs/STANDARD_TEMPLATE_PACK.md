# ELARA ZERO-OPS / DFN-0 — Standard Template Pack

## Purpose
Define the first reusable template pack for cascade application across the broader portfolio.

## Template 1 — Project Spec
Fields:
- Name
- Primary Family
- Optional Secondary Families
- Class (A/B/C/D)
- Primary Gap
- Current State
- What can be done now
- What requires external resources
- Output
- Risks
- Next Highest-Value Step

## Template 2 — Readiness Card
Fields:
- Readiness Score
- Reason for current score
- Main blocker
- Main enabler
- Expected next readiness jump

## Template 3 — Dependency Pack
Fields:
- Internal dependencies
- External dependencies
- Stack dependencies
- Operational dependencies
- Physical-world dependencies

## Template 4 — Reusable Assets Pack
Fields:
- Reusable artifacts
- Reusable documents
- Reusable workflow patterns
- Reusable runtime patterns
- Reusable security/policy rules

## Template 5 — Next-Step Pack
Fields:
- Current priority
- Immediate next step
- Why this is the highest-value move
- Expected output in the next stage

## Rule
These templates should be used to increase coherence, reuse, readiness tracking, and convertibility across the portfolio.
